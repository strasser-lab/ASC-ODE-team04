Exercise 18.4¶

In this exercise we had to:

  1. Add additional useful operators for the AutoDiff class

  2. Add some more functions (cos, exp, log, …) for the AutoDiff class.

  3. Evaluate and plot Legendre-polynomials up to order 5, in the interval −1≤x≤1-1 \le x \le 1−1≤x≤1. Evaluate and plot also their derivatives (using AutoDiff).

Implemented operators/functions (subtask 1, 2)¶

We implemented some useful operators and functions for the AutoDiff class in the autodiff.hpp.

Arithmetic operators¶

We have: u(x)\\ u(x)u(x) and its derivative u′u' \\u′ v(x)v(x)v(x) and its derivative v′v' \\v′

Then,

Addition¶

Mathematics:

f(x)=u(x)+v(x),f′(x)=u′+v′f(x) = u(x) + v(x), \quad f'(x) = u' + v'f(x)=u(x)+v(x),f′(x)=u′+v′
(1)

Example:

f(x)=x2+3x⇒f′(x)=2x+3f(x) = x^2 + 3x \Rightarrow f'(x) = 2x + 3f(x)=x2+3x⇒f′(x)=2x+3
(2)

In the code the implementation is:

template <typename T>
AutoDiff<T> operator+(const AutoDiff<T>& a, const AutoDiff<T>& b) {
    return AutoDiff<T>(a.val + b.val, a.der + b.der);
}
Subtraction¶

Mathematics:

f(x)=u(x)−v(x),f′(x)=u′−v′f(x) = u(x) - v(x), \quad f'(x) = u' - v'f(x)=u(x)−v(x),f′(x)=u′−v′
(3)

Example:

f(x)=x2−sin⁡(x)⇒f′(x)=2x−cos⁡(x)f(x) = x^2 - \sin(x) \Rightarrow f'(x) = 2x - \cos(x)f(x)=x2−sin(x)⇒f′(x)=2x−cos(x)
(4)

In the code the implementation is:

template <typename T>
AutoDiff<T> operator-(const AutoDiff<T>& a, const AutoDiff<T>& b) {
    return AutoDiff<T>(a.val - b.val, a.der - b.der);
}
Multiplication¶

Mathematics:

f(x)=u(x)⋅v(x),f′(x)=u′v+uv′f(x) = u(x) \cdot v(x), \quad f'(x) = u'v + uv'f(x)=u(x)⋅v(x),f′(x)=u′v+uv′
(5)

Example:

f(x)=x⋅sin⁡(x)⇒f′(x)=sin⁡(x)+xcos⁡(x)f(x) = x \cdot \sin(x) \Rightarrow f'(x) = \sin(x) + x\cos(x)f(x)=x⋅sin(x)⇒f′(x)=sin(x)+xcos(x)
(6)

In the code the implementation is:

template <typename T>
AutoDiff<T> operator*(const AutoDiff<T>& a, const AutoDiff<T>& b) {
    return AutoDiff<T>(a.val * b.val, a.der * b.val + a.val * b.der);
}
Division¶

Mathematics:

f(x)=u(x)v(x),f′(x)=u′v−uv′v2f(x) = \frac{u(x)}{v(x)}, \quad f'(x) = \frac{u'v - uv'}{v^2}f(x)=v(x)u(x)​,f′(x)=v2u′v−uv′​
(7)

Example:

f(x)=xx+1⇒f′(x)=1(x+1)2f(x) = \frac{x}{x + 1} \Rightarrow f'(x) = \frac{1}{(x + 1)^2}f(x)=x+1x​⇒f′(x)=(x+1)21​
(8)

In the code the implementation is:

AutoDiff<T> operator/(const AutoDiff<T>& a, const AutoDiff<T>& b) {
    T bv = b.val;
    return AutoDiff<T>(a.val / bv,
                       (a.der * bv - a.val * b.der) / (bv * bv));
}
Negative sign¶

Mathematics:

f(x)=−u(x),f′(x)=−u′f(x) = -u(x), \quad f'(x) = -u'f(x)=−u(x),f′(x)=−u′
(9)

Example:

f(x)=cos⁡(x)⇒f′(x)=sin⁡(x)f(x) = \cos(x) \Rightarrow f'(x) = \sin(x)f(x)=cos(x)⇒f′(x)=sin(x)
(10)

In the code the implementation is:

template <typename T>
AutoDiff<T> operator-(const AutoDiff<T>& a) {
    return AutoDiff<T>(-a.val, -a.der);
}

Elementary functions¶

Here we implemented the following useful elementary functions. We have: u(x)\\ u(x)u(x) and its derivative u′u' \\u′

Then,

Sinus¶

Mathematics:

f(x)=sin⁡(u(x))⇒f′(x)=cos⁡(u(x))⋅u′(x)f(x) = \sin(u(x)) \Rightarrow f'(x) = \cos(u(x)) \cdot u'(x)f(x)=sin(u(x))⇒f′(x)=cos(u(x))⋅u′(x)
(11)

Example:

f(x)=sin⁡(x)⇒f′(x)=cos⁡(x)f(x) = \sin(x) \Rightarrow f'(x) = \cos(x)f(x)=sin(x)⇒f′(x)=cos(x)
(12)

In the code the implementation is:

template <typename T>
AutoDiff<T> sin(const AutoDiff<T>& a) {
    return AutoDiff<T>(std::sin(a.val), std::cos(a.val) * a.der);
}
Cosinus¶

Mathematics:

f(x)=cos⁡(u(x))⇒f′(x)=−sin⁡(u(x))⋅u′(x)f(x) = \cos(u(x)) \Rightarrow f'(x) = -\sin(u(x)) \cdot u'(x)f(x)=cos(u(x))⇒f′(x)=−sin(u(x))⋅u′(x)
(13)

Example:

f(x)=cos⁡(x)⇒f′(x)=−sin⁡(x)f(x) = \cos(x) \Rightarrow f'(x) = -\sin(x)f(x)=cos(x)⇒f′(x)=−sin(x)
(14)

In the code the implementation is:

template <typename T>
AutoDiff<T> cos(const AutoDiff<T>& a) {
    return AutoDiff<T>(std::cos(a.val), -std::sin(a.val) * a.der);
}
Exponential¶

Mathematics:

f(x)=eu(x)⇒f′(x)=eu(x)⋅u′(x)f(x) = e^{u(x)} \Rightarrow f'(x) = e^{u(x)} \cdot u'(x)f(x)=eu(x)⇒f′(x)=eu(x)⋅u′(x)
(15)

Example:

f(x)=ex⇒f′(x)=exf(x) = e^x \Rightarrow f'(x) = e^xf(x)=ex⇒f′(x)=ex
(16)

In the code the implementation is:

template <typename T>
AutoDiff<T> exp(const AutoDiff<T>& a) {
    T ev = std::exp(a.val);
    return AutoDiff<T>(ev, ev * a.der);
}
Logarithm¶

Mathematics:

f(x)=ln⁡(u(x))⇒f′(x)=u′(x)u(x)f(x) = \ln(u(x)) \Rightarrow f'(x) = \frac{u'(x)}{u(x)}f(x)=ln(u(x))⇒f′(x)=u(x)u′(x)​
(17)

Example:

f(x)=ln⁡(ex)=x⇒f′(x)=1ex⋅ex=1f(x) = \ln(e^x) = x \Rightarrow f'(x) = \frac{1}{e^x} \cdot e^x = 1f(x)=ln(ex)=x⇒f′(x)=ex1​⋅ex=1
(18)

In the code the implementation is:

template <typename T>
AutoDiff<T> log(const AutoDiff<T>& a) {
    return AutoDiff<T>(std::log(a.val), a.der / a.val);
}
Power¶

Mathematics:

f(x)=(u(x))c⇒f′(x)=c⋅(u(x))c−1⋅u′(x)f(x) = (u(x))^c \Rightarrow f'(x) = c \cdot (u(x))^{c-1} \cdot u'(x)f(x)=(u(x))c⇒f′(x)=c⋅(u(x))c−1⋅u′(x)
(19)

Example:

f(x)=x3⇒f′(x)=3x2f(x) = x^3 \Rightarrow f'(x) = 3x^2f(x)=x3⇒f′(x)=3x2
(20)

In the code the implementation is:

template <typename T>
AutoDiff<T> pow(const AutoDiff<T>& a, T exponent) {
    T v = std::pow(a.val, exponent);
    T factor = exponent * std::pow(a.val, exponent - 1);
    return AutoDiff<T>(v, factor * a.der);
}

Mathematics:

In the code the implementation is:

template <typename T, typename S>
AutoDiff<T> operator/(const AutoDiff<T>& a, S scalar)
{
    return AutoDiff<T>(a.val / scalar, a.der / scalar);

template <typename T, typename S>
AutoDiff<T> operator/(S scalar, const AutoDiff<T>& a)
{
    T g  = a.val;
    T g2 = g * g;
    return AutoDiff<T>(scalar / g, -scalar * a.der / g2);

Evaluate and plot Legendre-polynomials (and their derivatives) up to order 5, in the interval −1≤x≤1-1 \le x \le 1−1≤x≤1 (subtask 3)¶

We implemented this process in main.cpp. It evaluates in the interval −1≤x≤1-1 \le x \le 1−1≤x≤1 the Legendre-polynomials up to order 5 and their derivatives (using AutoDiff). For the plot, it generates a legendre.csv file that can be plotted using plot_legendre.py program.

Evaluation¶

The Legendre polynomials are defined using the following recursive form:

P0=1P1=xPk(x)=(2k−1)xPk−1(x)−(k−1)Pk−2(x)k,k≥2\begin{aligned} P_{0} = 1 \\ P_{1} = x \\ P_{k}(x) = \frac{(2k - 1)xP_{k-1}(x) - (k - 1)P_{k-2}(x)}{k}, \qquad k \ge 2 \end{aligned}P0​=1P1​=xPk​(x)=k(2k−1)xPk−1​(x)−(k−1)Pk−2​(x)​,k≥2​
(34)

The program evaluates P, P′P, \, P'P,P′ for 400 (xxx) points in the interval [−1,1][-1,1][−1,1]. So it computes the values of the Legendre polynomials, and the values of their derivatives for each point. Then the program writes the results into the legendre.csv file in the following format:

x,P0,P0′,P1,P1′,P2,P2′,P3,P3′,P4,P4′,P5,P5′x, \quad P_0, \quad P_{0}', \quad P_1, \quad P_{1}', \quad P_2, \quad P_{2}', \quad P_3, \quad P_{3}', \quad P_4, \quad P_{4}', \quad P_5, \quad P_{5}'x,P0​,P0′​,P1​,P1′​,P2​,P2′​,P3​,P3′​,P4​,P4′​,P5​,P5′​
(35)

For example a line looks like this:

0.5,1.0,0.0,0.5,1.0,−0.375,3.0,0.625,7.5,−0.625,−15.0,−1.875,−18.750.5, \quad 1.0, \quad 0.0, \quad 0.5, \quad 1.0, \quad -0.375, \quad 3.0, \quad 0.625, \quad 7.5, \quad -0.625, \quad -15.0, \quad -1.875, \quad -18.750.5,1.0,0.0,0.5,1.0,−0.375,3.0,0.625,7.5,−0.625,−15.0,−1.875,−18.75
(36)

(So for instance: P5′(0.5)=−18.75P'_{5}(0.5) = -18.75P5′​(0.5)=−18.75)

In the end it also prints in the command window, whether the legendre.csv was created succesfully.

Exercise 18.5 (Pendulum)¶

In this section, we demonstrate how to implement differential equations without explicitly providing the derivatives (see, for example, Exercise 17.5.1). To achieve this, we introduce the AutoDiff class, which handles the differentiation automatically.

For the pendulum, the governing second-order ODE is:

α′′=−glsin⁡(α),α(t0)=π+0.001,  α′(t0)=0,\alpha^{\prime \prime} = -\frac{g}{l} \sin (\alpha), \qquad \alpha(t_0) = \pi +0.001, \; \alpha^\prime(t_0) = 0,α′′=−lg​sin(α),α(t0​)=π+0.001,α′(t0​)=0,
(37)

Our results are consistent with previous conclusions

  • Explicit Euler accumulates numerical errors and introduces artificial energy into the system. This leads to an overestimation of the amplitude.

  • Crank–Nicolson behaves as expected: the oscillations remain stable and physically correct, even with only a tiny perturbation in the initial angle.
